# VaultSpeed Labs

VaultSpeed Labs is the home for open, community-driven tools — and for productized 
accelerators built by VaultSpeed Professional Services to address recurring customer needs.

Every accelerator here is designed to reduce delivery risk, improve reuse across projects 
and platforms, and accelerate time-to-value — without turning every customer need into 
bespoke tooling.


## Questions?

Have a question about one of these tools or want to explore how they fit your project?
Reach out to the VaultSpeed team at [vaultspeed.com/contact](https://www.vaultspeed.com/contact).

## Template and Metadata Portability

*Reduce rework, prevent lock-in, and support reuse.*

- **Template Copier** — Exports and imports templates, including settings, between projects 
  or subscriptions to support parallel development and reduce manual migration effort.
- **Source Import / Export** — Exports source metadata across releases into a mutable YAML 
  format and imports it into other projects or subscriptions to support technology migration.

---

## Guided Configuration and Adoption Acceleration

*Reduce onboarding friction and late-stage rework.*

- **Source Wizard** — A guided configuration experience that helps teams set key parameters 
  and apply validated settings to the subscription.

---

## Orchestration and Runtime Optimization

*Improve scalability, parallelism, and operational independence.*

- **Business Vault Split for Airflow and Snowflake** — Splits monolithic Business Vault 
  orchestration into independently runnable units based on configuration.
- **Agent + Airflow + Postgres Docker Setup** — A ready-to-run Docker configuration for 
  agent and orchestration setup that reduces environment complexity.

---

## Governance, Quality, and Data Product Enablement

*Increase trust, observability, and data product readiness.*

- **Data Contract Generator** — Generates import-ready contracts from VaultSpeed metadata — 
  including tables, types, and lineage to Raw Data Vault — in JSON, YAML, PDF, or LaTeX.
- **Data Quality Framework** — Generates and executes foundational data quality checks, such 
  as referential integrity, duplicates, and missing ghost records, and stores results in a 
  Data Vault structure with support for custom soft rules.

---

## Deployment and CI/CD Enablement

*Improve release control, flexibility, and safety.*

- **Artifact Builder** — Packages selected generations — including deltas, new FMCs, and 
  custom objects — into a deployable artifact with file exclusion, pre/post hooks, and 
  YAML-driven execution order.
